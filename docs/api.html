GSAP Horizontal Loop — API Reference

Production-ready API docs for the horizontal loop carousel component (source: README).

Overview

A seamless, responsive, horizontally looping carousel built on GSAP. Features touch/mouse dragging, autoplay, navigation, pagination, accessibility, and a compact programmatic API.

Installation

Install GSAP via npm or include it from a CDN. Draggable and Inertia plugins are optional for advanced dragging/momentum. 

// npm
npm install gsap

// or include GSAP via CDN (example)
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.0/gsap.min.js"></script>

Quick Start

<!-- markup -->
<div class="carousel-container">
  <div class="carousel-item">Slide 1</div>
  <!-- more slides -->
</div>

<script>
const carousel = horizontalLoop('.carousel-container', {
  speed: 1,
  gap: '20px',
  draggable: true,
  autoplayDelay: 3,
});
</script>

Configuration Options

The component accepts a single options object. The table below lists the most-used options.

Option Description (default)
responsive Object|null — breakpoint mapping (null)
speed number — speed multiplier (1)
gap string — CSS gap between items ("0px")
draggable boolean — enable dragging (false)
repeat number — number of loop repeats (0, -1 for infinite)
paused boolean — start paused (true)
autoplayDelay number — seconds between advances (0 disables)
reversed boolean — reverse direction (false)
prevNav / nextNav Element|string|Array — navigation control selectors
dots Element|string|Array — pagination container
snap number|false — snap increment (1)
onChange Function|null — called when visible slide changes
onInitialized Function|null — called after initialization
ariaLabel string — ARIA label ("Carousel")
accessibilityEnabled boolean — enable accessibility (true)

Core Methods

The API exposes convenient control methods for programmatic usage.

  • toIndex(index, options?) — Navigate to a slide.
  • next(options?) / previous(options?) — Step forward/back.
  • current() — Returns current index.
  • closestIndex(setCurrent?) — Closest slide index.
  • playAutoplay() / pauseAutoplay() — Autoplay control.
  • refresh(deep?) — Recalculate layout.
  • cleanup() — Destroy instance and listeners.

Callback Data

{
  currentItem: HTMLElement,
  currentIndex: number,
  totalItems: number,
  progress: number,           // 0..1
  slideWidth: number,
  timeline: GSAPTimeline,
  config: Object
}

Accessibility & Keyboard

ARIA attributes, keyboard controls, screen reader announcements and focus management are implemented. Standard keys:

  • Arrow ← / → : Navigate slides
  • Home / End : First / Last slides
  • Enter / Space : Activate controls

Check the examples directory for basic and advanced setups. Common issues (slides not visible, dragging not working, autoplay not starting) and remedies are documented in the README.
