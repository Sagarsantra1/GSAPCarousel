GSAP Carousel

A smooth, performant carousel library built with GSAP. Minimal, accessible, and easy to integrate.

Download (minified) Download (dev) Read the API View examples

Prefer a CDN? Use the minified file hosted on GitHub Pages in your project (replace username if hosted elsewhere): 

<script src="https://Sagarsantra1.github.io/GSAPCarousel/dist/gsap-carousel.min.js"></script>

Installation

Files

  • dist/gsap-carousel.js — development (readable) build
  • dist/gsap-carousel.min.js — production minified build

Quick start

Include GSAP (and plugins if needed) before the carousel script. Then initialize with a selector and options. 

<!-- include GSAP (cdn) -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.0/gsap.min.js"></script>
<!-- then the carousel -->
<script src="/dist/gsap-carousel.min.js"></script>

<!-- markup -->
<div class="carousel-container">
  <div class="carousel-item">Item 1</div>
  <!-- more items -->
</div>

<script>
const carousel = horizontalLoop('.carousel-container', {
  speed: 1,
  gap: '20px',
  draggable: true
});
</script>

Notes

  • For drag momentum include the Draggable + InertiaPlugin (GSAP Club required for Inertia).
  • Serve the examples folder with a local server (e.g. `npx http-server examples`) to avoid CORS issues when loading assets.
